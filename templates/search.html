Инструкция по созданию запроса

  1. Слово в любой форме без кавычек - поиск по лемме. Пример: маме => находится всё по лемме "мама".
  2. Слово в одинарных кавычках - поиск по конкретной словоформе. Пример: 'маму' => находится только словоформа "маму".
  3. Поиск по части речи: upos=ЧАСТЬ_РЕЧИ. Пример: upos=NOUN найдёт все предложения с существительными.
  4. Поиск одной (только одной в запросе) из других морфологических характеристик (падеж, число, время и др.): features=ЗНАЧЕНИЕ_ФИЧИ. Пример: features=Acc найдёт все слова, стоящие в винительном падеже.
  5. Слово + его характеристика: слово+upos=ЧАСТЬ_РЕЧИ или слово+features=ЗНАЧЕНИЕ_ФИЧИ. Пример: мама+features=Acc.
  6. Два и более слов в запросе записываются через пробел.
